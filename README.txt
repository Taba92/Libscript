Libreria di utilizzo generico per lo scripting.
Gestisce:
    -) Logging
    -) Configurazioni scripting
    -) Backup

HINT:
    Installazione di pacchetti locali alla libreria di scripting:
        pip install 'nomepacchetto' -t Pacchetti/'nomepacchetto'

    Installazione della libreria tramite pip:
        pip install 'link/github/Libscript'
